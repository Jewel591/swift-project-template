# PRD

# 产品概述

- 产品名称：Talon
- 一句话定位：Eagle 资源库的 iOS 移动端伴侣 — AI 驱动、本地优先、隐私零风险
- 平台：iOS 18+ / macOS 15+ (SwiftUI)
- 框架：SwiftUI + MVVM
- 分发：App Store
- 后端：Supabase
- 数据持久化：SwiftData

## 产品背景

Eagle 拥有 40 万+ 用户却没有移动端，其 5.0 版本全力押注桌面端 AI 功能，官方明确表示短期内不会开发移动版本。iOS 端没有任何一款产品能同时满足「专业素材管理 + 多格式支持 + 标签系统 + Eagle 资源库兼容」的需求组合。

现有唯一第三方 Eagle Viewer 仅提供只读浏览，留下了从编辑、AI 搜索到素材添加的巨大价值空间。

## 目标用户

### 核心用户

已在桌面端使用 Eagle 的 UI/UX 设计师、平面设计师、插画师（约 5-20 万人）。典型场景：

- 通勤时在手机上刷到好设计，想立即保存到 Eagle 库
- 开会时向客户展示参考素材
- iPad 上用 Procreate 时需要参考图
- 碎片时间整理未分类素材

### 扩展用户

- 使用 iPad + Apple Pencil 的数字艺术家
- 需要移动端管理参考图的自由职业设计师

### 长尾用户

- 摄影师（管理大量图片素材）
- 建筑/室内设计师（材质和灵感管理）
- 内容创作者

## 核心价值主张

在 iPhone 和 iPad 上浏览、搜索和管理你的 Eagle 素材库，用 AI 让整理效率提升 10 倍，所有数据永不离开你的设备。

### 三大差异化支柱

1. Eagle 原生兼容：唯一能完整读写 Eagle `.library` 格式的 iOS App，无需导入转换，打开 iCloud Drive 里的 Eagle 库即可使用
2. AI 本地智能：MobileCLIP 语义搜索 + 自动标签 + OCR + 颜色搜索，全部设备端处理，素材隐私零风险
3. Apple 生态深度集成：iCloud 同步、Shortcuts/Siri 集成、Apple Pencil 标注、iPad 多窗口/Stage Manager、Share Extension

## 商业模式

| 层级 | 价格 | 核心权益 |
|------|------|---------|
| 免费版 | $0 | 浏览 Eagle 资源库（限 1 个库）、基础搜索和筛选、50 张/月 AI 搜索配额、基础预览 |
| Pro 年付 | $2.99/月（$29.99/年） | 无限库、无限 AI 搜索/标签、OCR 搜索、颜色搜索、高级筛选、编辑/添加素材、Share Extension |
| Pro 月付 | $4.99/月 | 同上 |
| 终身版（限时） | $49.99 一次性 | 所有 Pro 功能永久解锁 |

冷启动策略：首发 30 天内 Pro 年付 $19.99（限时 6 折），终身版 $39.99（限时 5 折）。

# 基础（P0 — MVP 首版发布）

## F-001 Eagle .library 资源库解析与浏览

### 描述

完整解析 Eagle 桌面端的 `.library` 文件夹结构，在 iOS 端还原文件夹层级、素材元数据、标签系统等全部信息。

### Eagle .library 目录结构

```
YourLibrary.library/
├── images/
│   ├── LSVNSY7DPP56X.info/     # 每个素材一个文件夹（13位随机ID）
│   │   ├── metadata.json        # 元数据（标签、颜色、尺寸、来源URL等）
│   │   ├── image1.jpg           # 原始文件
│   │   └── _thumbnail.png       # Eagle生成的缩略图
├── metadata.json                # 全局元数据
└── tags.json                    # 标签数据
```

### 需要解析的元数据字段

每个素材的 `metadata.json` 包含：

- 唯一 ID
- 名称
- 文件大小
- 时间戳（创建/修改/导入时间）
- 文件扩展名
- 标签数组
- 所属文件夹 ID（支持多文件夹归属）
- 来源 URL
- 注释/备注
- 图片尺寸（宽 × 高）
- 星级评分（0-5）
- 自动提取的颜色调色板（palettes）

### 功能要求

- 支持打开和解析完整的 `.library` 目录结构
- 还原多层级文件夹树形结构
- 解析全局 `tags.json` 构建标签索引
- 解析每个素材的 `metadata.json` 提取完整元数据
- 使用 Eagle 预生成的 `_thumbnail.png` 作为缩略图，避免重复生成
- 支持同一素材归属多个文件夹（Eagle 的链接机制）
- 首次打开资源库时建立本地索引（SwiftData），后续增量更新

### 验收标准

- 能正确打开包含 1000+ 素材的 Eagle 资源库
- 文件夹层级结构与 Eagle 桌面端完全一致
- 素材元数据（标签、评分、颜色、备注）100% 正确解析
- 首次索引 1000 张素材不超过 10 秒

## F-002 iCloud Drive / SMB 网络共享读取

### 描述

支持通过 iCloud Drive 和 SMB 网络共享访问 Eagle 桌面端同步的 `.library` 资源库，实现跨设备无缝访问。

### 功能要求

- 支持通过 iOS Files App 的 Document Picker 选择 `.library` 目录
- 支持 iCloud Drive 中的 Eagle 资源库（桌面端通过 iCloud 同步的场景）
- 支持 SMB 网络共享中的 Eagle 资源库（局域网访问桌面端 Eagle 库）
- 处理 iCloud 文件的按需下载（`startDownloadingUbiquitousItem`）
- 记住最近打开的资源库路径，支持快速切换
- 监听文件系统变更，检测桌面端的增量修改

### 验收标准

- 能通过 Document Picker 成功选择 iCloud Drive 中的 Eagle 资源库
- 能通过 SMB 连接局域网中的 Eagle 资源库
- iCloud 文件按需下载流程流畅，有明确的下载进度提示
- 支持至少保存 5 个最近打开的资源库路径

## F-003 瀑布流/网格浏览

### 描述

提供多种布局方式浏览素材，还原 Eagle 桌面端的视觉浏览体验。

### 功能要求

- 瀑布流布局（Waterfall/Masonry）：根据图片原始比例自适应排列
- 网格布局（Grid）：等尺寸方格排列，适合快速扫描
- 列表布局（List）：带元数据信息的紧凑列表
- 布局模式切换：支持在三种布局间一键切换
- 缩略图尺寸调节：支持拖动滑块调整缩略图大小（小/中/大/超大）
- 长按预览：长按素材显示大图预览 + 基础元数据
- 3D Touch / Haptic Touch 预览（Peek & Pop）
- 滚动性能：10000+ 素材流畅滚动，无卡顿
- 下拉刷新：同步最新的资源库变更

### iPad 适配

- 支持 iPad 的大屏幕多列布局
- 根据屏幕宽度动态调整列数
- 支持横竖屏自适应

### 验收标准

- 瀑布流布局图片无裁切、无变形
- 10000 张素材滚动帧率 ≥ 55fps
- 布局切换动画流畅（< 300ms）
- 长按预览响应时间 < 200ms

## F-004 标签搜索和多维筛选

### 描述

提供多维度搜索和筛选能力，帮助用户在大量素材中快速定位目标。

### 搜索功能

- 关键字搜索：搜索素材名称、标签、备注、来源 URL
- 标签搜索：支持标签名模糊匹配、标签联合筛选（AND/OR）
- 搜索建议：输入时实时显示匹配的标签和历史搜索
- 搜索历史：记录最近 50 条搜索记录
- 搜索速度：1 万张素材搜索响应 < 0.5 秒

### 筛选器

- 文件类型筛选：图片/设计文件/视频/音频/字体/3D/文档
- 标签筛选：支持多标签组合筛选
- 星级评分筛选：按星级过滤（≥ 1 星、≥ 2 星 …… 5 星）
- 颜色筛选：按主色调筛选（从素材 palette 数据）
- 时间范围筛选：按导入时间/修改时间筛选
- 文件大小筛选：按文件大小区间筛选
- 尺寸筛选：按图片宽高筛选（横版/竖版/方形）
- 来源筛选：按来源网站域名筛选

### 排序

- 按导入时间排序（默认，最新优先）
- 按修改时间排序
- 按名称排序（A-Z / Z-A）
- 按文件大小排序
- 按星级评分排序
- 按颜色排序

### 验收标准

- 搜索结果实时更新，输入停顿 300ms 后触发搜索
- 多维筛选可叠加使用
- 筛选状态清晰可见，支持一键清除全部筛选

### 描述

支持 Eagle 管理的多种文件格式在 iOS 端预览，至少覆盖最常见的设计文件格式。

- 颜色调色板展示：显示 Eagle 提取的主色调
- 分享：系统分享菜单导出原始文件
- 复制：长按复制图片到剪贴板
- 用其他 App 打开：Quick Look 联动系统注册的 App

## F-006 离线缓存

### 描述

智能预加载缩略图和常用素材，确保断网时仍可浏览和搜索。

### 功能要求

- 缩略图缓存：首次打开资源库时下载/缓存所有缩略图到本地
- 原图按需缓存：用户查看过的原始文件自动缓存
- 智能预加载：预加载当前浏览位置前后的原始文件
- 收藏素材离线：标记为收藏的素材自动缓存原始文件
- 缓存容量管理：用户可设置最大缓存空间（默认 2GB，可调 500MB - 10GB）
- LRU 缓存淘汰：空间不足时自动清理最久未访问的缓存
- 缓存状态指示：未缓存的素材显示云端图标
- 元数据始终可用：标签、评分、备注等元数据存入 SwiftData，始终离线可用

### 验收标准

- 断网状态下可浏览已缓存的素材
- 断网状态下搜索和筛选功能正常（基于本地索引）
- 缓存空间不超过用户设置的上限
- 缓存清理不影响用户数据

# 增强（P1 — 首版后 1-2 个月更新）

## F-101 AI 语义搜索

### 描述

基于 MobileCLIP 模型实现自然语言搜图，用户可用日常语言描述想找的素材。

### 功能要求

- 自然语言输入：支持中英文描述（如「蓝色渐变的登录页设计」「极简风格的名片」）
- 模型选择：Apple MobileCLIP-S0（Core ML，约 150MB）
- 索引构建：首次使用时对所有素材生成 CLIP 向量（512 维），存储到本地
- 索引速度：约 2000 张/分钟（iPhone 12+）
- 搜索速度：1 万张素材搜索 < 1 秒
- 相似度排序：按余弦相似度降序展示结果
- 离线可用：模型和向量均存储在本地，无需网络
- 增量索引：新增素材自动追加向量，无需全量重建
- 搜索结果展示相似度分数（可选开关）

### 免费版限制

- 每月 50 次 AI 搜索配额
- Pro 版无限制

### 技术方案

- 将 CLIP 模型拆分为 TextEncoder 和 ImageEncoder 两个 Core ML 模型
- 图片向量预计算并持久化存储
- 文字向量实时计算
- 512 维向量余弦相似度比较

### 验收标准

- 中英文搜索查询均能返回语义相关结果
- 搜索「sunset landscape」能准确找到日落风景类素材
- 索引 5000 张素材不超过 3 分钟
- 搜索响应时间 < 1 秒

## F-102 AI 自动标签

### 描述

导入或首次扫描素材时自动生成语义标签，减少手动整理工作量。

### 功能要求

- Vision 分类：使用 `VNClassifyImageRequest` 识别 5000+ 物体类别
- 标签生成：为每张图片生成 3-8 个推荐标签
- 置信度筛选：仅展示置信度 > 70% 的标签
- 标签审核：用户可逐一确认/拒绝/编辑推荐标签
- 批量应用：支持选中多张素材一键应用推荐标签
- Foundation Models 增强（iOS 26+）：调用本地 LLM 生成更语义化的标签描述
- 标签映射：将 Vision 的英文分类标签映射为用户友好的中英文标签
- 已有标签保护：不覆盖用户在 Eagle 桌面端已设置的标签

### 技术方案

| 层级 | 方案 | 适用 |
|------|------|------|
| 基础层 | VNClassifyImageRequest | 5000+ 物体类别，零配置 |
| 增强层 | Foundation Models Framework | iOS 26+ 本地 LLM 语义标签 |
| 扩展层 | Create ML 自定义模型 | 设计风格分类（极简/扁平/拟物等） |

### 验收标准

- 自动标签准确率 ≥ 80%（人工抽检 100 张）
- 单张图片标签生成 < 100ms
- 不覆盖已有标签
- 用户确认流程清晰易懂

## F-201 图片去重检测

### 描述

### 功能要求

- 相似度阈值：用户可调节相似度阈值（默认 90%）
- 分组展示：将重复/相似素材分组展示
- 批量处理：支持批量保留/删除重复素材
- CLIP 语义去重：复用 AI 搜索的 CLIP 向量进行语义级去重

### 验收标准

- 精确去重准确率 100%
- 感知去重召回率 ≥ 90%，误报率 < 5%
- 1 万张素材去重扫描 < 2 分钟
